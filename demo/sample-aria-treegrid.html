Example: WAI-ARIA Tree Grid - Prototype

This Fancytree Tree Grid has uses a prototypical extension to explore ARIA support as discussed here:
w3c/aria-practices/treegrid/examples/treegrid/treegrid-1.html

Extended mode adds this behavior:

  • Embedded inputs are not focused by default (but Enter and Space still work on checkboxes). Instead there is a distinct 'input-mode' that must be activated by Enter or alphanum keys.
  • Enter: In row-mode: switch to cell-mode. In cell-mode: execute default action or focus embedded text input (i.e. switch to input-mode). In input-mode: switch back to cell-mode.
  • Esc: In cell-mode: switch to row-mode. In input-mode: switch to cell-mode.

See also the ARIA Tree View example.
