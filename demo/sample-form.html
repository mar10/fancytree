Example: Form

Two Fancytrees embedded in a form, together with some standard elements.
  • Use the TAB key to move the focus between form elements.
    Note that a tree behaves like a single control (like a radio button group)
  • Experimental support for WAI-ARIA
  • The submit handler is hooked to generate hidden checkbox elements for selected and active nodes.
    Note that in select mode 3 only the topmost nodes are considered.
    Click the Submit button at the bottom of this page to POST these values to the server (together with the other form elements).
